class Frankenstein.frankenstein.PulseEditorApp(root, layout=None, oper_path=None, on_add_operation=None, on_return=None)Â¶

Bases: object

Graphical editor for quantum control pulses and operations.

The PulseEditorApp is the Frankenstein GUI layer that lets you load a Livingston layout, create and visualize pulses (drive/flux/parametric), and export complete operation descriptions (.oper) that include both the layout and all pulse segments. It can also round-trip operations back to ArchBuilder and open the AIgor assistant for automatic synthesis.
