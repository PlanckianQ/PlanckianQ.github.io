User Guide
==========

Welcome to the **ArchBuilder User Guide**.

This manual will show you how to install, configure, and use the main features.

Installation
------------

1. Clone the repository.
2. Install dependencies:
   .. code-block:: bash

      pip install -r requirements.txt

Launching the App
-----------------

To start ArchBuilder, run:

.. code-block:: bash

   python ArchBuilder.py

Basic Usage
-----------

Once open, you can:
- Create a new project
- Add quantum components
- Simulate using the *Bovary* backend
