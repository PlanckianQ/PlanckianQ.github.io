get_frequencies(plot: bool = False, variable_component: str = '', component_values: ndarray = None, xscale: float = 1, yscale: float = 1e-09, title: str = '', xlabel: str = '', ylabel: str = '(GHz)', x_axis_vector: ndarray = None)¶

Returns the values of the frequency of the qubits of the system, when changing one parameter of the system. Optionally it plots them. The frequency for each qubit ‘n’ is computed as:

\[f_n = \omega_n - A_n - \sum_{m\neq n}\frac{\chi_{mn}}{2}\]

with \(\omega_n\) the frequency of the corresponding mode, \(A_n\) the anharmonicity and \(\chi_{mn}\) the cross-kerr interaction between modes ‘m’ and ‘m’.

Parameters:

  • plot (bool) – Wheter plot the values. Default is False

  • variable_component (str) – The variable component which is changed.

  • component_values (np.ndarray) – Values of the variable component.

  • n_stat (int) – The number of state considered for the qubits. Default is 2.

  • taylor (int) – Degree of the Taylor expansion of the cosine potential. Default is 4.

  • excitations (int) – Number of excitations considered in building the Hamiltonian. Default is 6.

  • xscale (float) – The scale of the x-axis.

  • yscale (float) – The scale of the y-axis.

  • xlabel (str) – The label of the x-axis (the measure unit).

  • ylabel (str) – The label of the y-axis (the measure unit). Default is ‘(GHz)’.

  • xaxis_vector (np.ndarray) – The vector on placed on the x-axis.

Returns:

  • freqs (dict) – A dict containing the values of frequencies. Each key is a Fock state (a Junction), when the circuit topology (neighbours attribute) is empty (not empty).

  • Author

  • ——

Steppenwolf.circuit_builder.sw_project_to_qubit(H_qobj: Qobj, N: int, M: int, H0_qobj: Qobj = None, eps: float = 1e-12) → Qobj¶

Proietta un’Hamiltoniana di N qubit (ciascuno troncato a M livelli) nello spazio computazionale |n1 … nN> con ni∈{0,1}, includendo gli effetti virtuali di Q (livelli >=2) al 2º ordine tramite la trasformazione di Schrieffer–Wolff (formula simmetrica a denominatori).

H_eff = P H P + δH^(2), con

δH_ab = 1/2 * sum_{m∈Q} H_{a m} H_{m b} [ 1/(E_a - E_m) + 1/(E_b - E_m) ],

dove E_k sono gli autovalori di H0 (scelto come base per i denominatori).

Parametri¶

H_qobjQobj

Hamiltoniana completa (dimensione D×D, D=M**N) nella base prodotto.

Nint

Numero di qubit fisici (sistemi).

Mint

Dimensione locale (troncamento per qubit/transmon).

H0_qobjQobj, opzionale

Parte “non perturbata” usata nei denominatori. Se None, usa diag(H).

epsfloat

Soglia per evitare divisioni per ~0 (regime non-dispersivo). Se un denominatore |E_p - E_q| < eps, viene sollevato un ValueError.

Ritorna¶

Qobj

Hamiltoniana efficace 2^N × 2^N nello spazio computazionale, simmetrizzata Hermitiana.
